🐍 Python (기술 트랙)

Python은 웹 백엔드, 자동화, 데이터 분석, AI, 스크립팅까지 넓게 쓰이는 동적 언어입니다. 여기서는 면접·실무에서 자주 묻는 핵심 — 실행 모델, 자료형, 함수와 스코프, 객체지향, 예외, 이터레이터/제너레이터, 표준 라이브러리, 패키징 — 을 "왜 그렇게 동작하는지"와 "어디서 실수하는지" 중심으로 정리합니다. 기준 버전은 Python 3.12입니다.

📚 학습 가이드 — 핵심 먼저, 그다음 심화

먼저 핵심 개념으로 큰 그림(실행 모델 → 자료형·함수 → OOP·예외 → lazy iteration → 표준 라이브러리·패키징)을 잡고, 깊이 알고 싶은 주제는 심화 / 상세로 들어가세요. 예: Python 언어 기본 → CPython 실행 모델, 함수와 스코프 → 데코레이터·메타프로그래밍.

🧩 핵심 개념

Python 실무·면접에서 기본기로 가장 자주 등장하는 우선순위 주제들.

언어와 실행 모델

Python 코드가 어떻게 실행되고, 이름과 타입이 어떻게 결정되는지부터 시작합니다.

  • Python 언어 기본 — 인터프리터·동적 타이핑·모듈CPython 기준 실행 흐름, bytecode, 동적 타이핑, 모듈/패키지 구조.
  • 자료형과 컬렉션 — list·tuple·dict·setmutability, slicing, comprehension, dict insertion order, 얕은/깊은 복사.
  • 함수와 스코프 — first-class function·closure·LEGB함수 객체, 클로저, LEGB 탐색, mutable default argument 함정.

객체와 제어 흐름

Python답게 객체를 만들고, 예외·컨텍스트·반복 흐름을 제어합니다.

  • 객체지향 Python — class·self·dunder·dataclass인스턴스/클래스 속성, 메서드 바인딩, dunder method, dataclass.
  • 예외와 컨텍스트 매니저 — try/except·with예외 계층, custom exception, try/finally, context manager protocol.
  • 이터레이터 / 제너레이터 — iterator protocol·yieldlazy evaluation, generator function/expression, StopIteration, 메모리 효율.

실무 기본 도구

프로젝트를 만들고, 파일을 다루고, 의존성을 관리하는 기본기입니다.

  • 파일 / 표준 라이브러리 — pathlib·json·datetime·collections·itertools실무에서 매일 쓰는 표준 도구와 안전한 파일 처리.
  • 가상환경과 패키징 — venv·pip·requirements·pyproject환경 격리, 의존성 고정, 패키지 구조, uv/poetry 개요.

🔬 심화 / 상세

핵심 페이지에서 분리한 깊이 있는 주제들. 면접에서 단독으로 깊이 묻는 항목들이라 페이지를 따로 둡니다. (작성 예정)

런타임 내부

  • CPython 실행 모델 작성 예정bytecode, frame object, reference counting, garbage collection.
  • GIL(Global Interpreter Lock) 작성 예정왜 필요한지, CPU-bound vs I/O-bound, multiprocessing/asyncio 대안.
  • asyncio 심층 작성 예정event loop, coroutine, task, await, cancellation, backpressure.

타입·메타프로그래밍·성능

  • 타입 힌트 심층 작성 예정mypy/pyright, Protocol, Generic, TypedDict, Pydantic.
  • 데코레이터와 메타프로그래밍 작성 예정function/class decorator, descriptor, metaclass.
  • 성능 최적화 작성 예정profiling, vectorization, C extension, PyPy.

도구와 데이터

  • 테스트와 품질 도구 작성 예정pytest fixture, monkeypatch, coverage, ruff, black.
  • 데이터 처리 기본 작성 예정pandas/numpy 핵심 구조, 벡터화, 메모리 주의점.
