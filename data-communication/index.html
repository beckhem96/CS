📡 데이터통신

데이터통신(Data Communication)은 데이터를 전기·빛·전파 같은 물리적 신호로 바꿔 보내고, 여러 장치가 같은 매체를 나눠 쓰며, 오류를 찾아내고 고치는 방법을 다루는 분야입니다. 네트워크가 "프로토콜과 계층"에 가깝다면, 데이터통신은 그 아래에서 실제 신호가 어떻게 이동하는지를 설명합니다.

📚 학습 가이드 — 핵심 먼저, 그다음 심화

먼저 핵심 개념을 한 번 훑어 신호·매체·오류의 큰 그림을 만든 뒤, 깊이 알고 싶은 주제는 같은 카테고리의 심화 / 상세로 들어가세요. 예: 변조 → OFDM / 오류 검출 → 샤논 한계.

🧩 핵심 개념

데이터통신을 처음 배우거나 면접 직전에 한 번 훑기 좋은 우선순위 개념들.

신호로 바꾸기

디지털 데이터를 실제 매체가 전달할 수 있는 신호로 표현하는 방법.

  • 변조(Modulation) AM/FM/PM, ASK/FSK/PSK/QAM — 반송파의 진폭·주파수·위상을 바꾸는 방법.
  • 신호 인코딩(Line Coding) NRZ, Manchester, 4B/5B, 8B/10B — 0과 1을 전압/전이 패턴으로 표현.

매체 나눠 쓰기

하나의 전송 매체를 여러 사용자·채널이 효율적으로 공유하는 방법.

  • 다중화(Multiplexing) TDM/FDM/WDM/CDM — 시간·주파수·파장·코드로 여러 신호를 합쳐 보내기.
  • 매체 접근 제어(MAC) CSMA/CD와 CSMA/CA — 공유 매체에서 충돌을 감지하거나 피하는 규칙.

오류 다루기

잡음이 있는 채널에서 잘못 전달된 비트를 발견하고, 가능한 경우 고치는 방법.

  • 오류 검출 / 정정 패리티, 체크섬, CRC, 해밍 코드 — 오류를 찾아내고 일부는 복구하기.

🔬 심화 / 상세

핵심 개념을 한 번 본 뒤, 신호 처리·무선 통신에서 깊이 물어볼 만한 주제를 별도 페이지로 정리합니다. (작성 예정)

채널과 무선 심화

  • 샤논 한계와 채널 용량 작성 예정 Shannon-Hartley, 잡음·대역폭·SNR의 관계.
  • OFDM 작성 예정 Wi-Fi/LTE/5G의 핵심 변조 방식, 부반송파, 가드 인터벌.
  • MIMO / 빔포밍 작성 예정 다중 안테나, 공간 다중화, 방향성 신호 제어.
  • 5G 핵심 기술 작성 예정 mmWave, Network Slicing, URLLC, massive MIMO.
