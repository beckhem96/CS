🏥 심평원(HIRA) 면접 대비

자기소개서 4개 문항을 기반으로 면접관이 던질 가능성이 높은 경험 deep dive 질문과 그 안에서 자연스럽게 묻기 좋은 연관 CS 개념 질문, 그리고 답변 가이드를 정리합니다. 각 페이지는 자소서 원문 → 예상 질문 → 답변 → 꼬리 질문 순으로 구성되어 있어, 말로 설명하는 연습용으로 그대로 읽으면서 사용할 수 있습니다.

🎯 활용 방법

  • 각 자소서 항목 페이지를 열고, 답변을 보지 않은 상태에서 질문만 보고 30초 안에 답변 흐름을 만들어 보세요.
  • 답변을 본 뒤에는 자기 답변과 비교해 비어 있는 키워드를 메모해 둡니다.
  • 연관 CS 개념 질문이 막히면 같은 페이지의 "관련 CS 노트" 링크로 이동해 다시 학습.

📝 자소서 항목별 예상 질문

1. 직무 지식·기술 습득 경험

BASS 통계 대시보드 1분→1초 단축, 외부 시험 적재 API, SSAFY 책크인 하이브리드 추천(우수상).

  • MyBatis + concat/cast Full Scan · PlatformTransactionManager · FastAPI DBeaver vs 서버 실행 계획 교차 검증, Selectivity 추정 실패, 학생 단위 트랜잭션 분리, 콘텐츠+협업 하이브리드 추천.

2. 관행을 깨고 원칙을 지킨 경험

ATV 자막 — 협력사 STT 지연 → CLOVA Speech API PoC → gRPC 양방향 스트리밍 + @Synchronized 임계영역.

  • CLOVA Speech · proto oneof · @Synchronized · lockCanvas 재시도 512byte PCM 청크, 시스템 앱 권한(SurfaceFlinger), 채널 변경 시 Race Condition, 3×23 그리드 버퍼.

3. 단점으로 인한 소통 어려움 극복 경험

Spring Batch 단독 결정 → 인프라 제약 미확인 → Crontab + Java 메모리 정렬 + 500건 청크로 재구현.

  • Spring Batch · Crontab · 의사결정 공유 · 게이미피케이션 순위 인프라 제약, Job/Step 구조, DB 부하 오프로딩, OOM 방지 청크, ADR과 사전 합의 습관.

4. 효율적 방식 도입 경험

BASS Properties 복호화 유틸리티 사내 배포 → 팀 구두 문의 0, 개발 흐름 단절 제거.

  • 대칭키 AES · KMS · Salt/Pepper · 시크릿 관리 CLI 형태 단일 jar 배포, 키 OS 권한 분리, Jasypt/Vault 비교, GCM·해싱·키 로테이션.
