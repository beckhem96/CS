☕ Java (기술 트랙)

Java는 백엔드·안드로이드·데이터 인프라 등 산업 전반에서 가장 오래 자리잡고 있는 JVM 언어입니다. 여기서는 면접·실무에서 깊이 묻는 핵심 — 언어 모델, OOP 문법, 예외, 컬렉션, 제네릭, 람다·Stream, JVM 메모리, 멀티스레딩 — 을 "왜 그렇게 설계됐는가"와 "어디서 실수하는가" 중심으로 정리합니다. 기준 버전은 Java 21 (LTS)입니다.

📚 학습 가이드 — 핵심 먼저, 그다음 심화

먼저 핵심 개념으로 큰 그림(언어가 어떻게 실행되는지 → OOP/예외 → 컬렉션·제네릭 → 람다·Stream → JVM·동시성)을 잡고, 깊이 알고 싶은 주제는 심화 / 상세로 들어가세요. 예: 멀티스레딩 기초 → JMM·Lock·CompletableFuture 심층, JVM 메모리 → GC 알고리즘 심층.

🧩 핵심 개념

Java 백엔드 면접에서 거의 매번 등장하는 우선순위 주제들. 처음 보는 사람도 이 8개만 끝까지 읽으면 큰 그림은 잡힙니다.

언어와 실행 모델

"내 코드가 어떻게 컴파일되고 실행되는가"부터 시작합니다.

  • Java 언어 기본 — JVM·JDK·JRE, 컴파일·실행 흐름 .java → bytecode → JVM 실행, 클래스 로딩 단계, 정적 타입과 플랫폼 독립성.
  • 객체지향과 Java 문법 — class / interface / record / enum 접근 제어자, 오버로딩 vs 오버라이딩, abstract와 인터페이스, record·enum의 쓰임.
  • 예외 처리 — checked vs unchecked, try-with-resources Throwable 계층, 예외 설계 원칙, 자원 자동 해제, 스택 트레이스 읽기.

표준 라이브러리의 핵심

실무 코드 대부분이 이 둘로 짜입니다.

  • 컬렉션 프레임워크 — List·Set·Map·Queue ArrayList vs LinkedList, HashMap의 충돌·resize·treeify, fail-fast Iterator.
  • 제네릭(Generic) — 타입 소거, bounded wildcard, PECS 컴파일 타임 타입 검사, 런타임 타입 소거, Producer Extends / Consumer Super.
  • 람다와 Stream API — 함수형 인터페이스·lazy evaluation Function/Predicate/Consumer/Supplier, 메서드 레퍼런스, 중간/최종 연산, 병렬 스트림 함정.

런타임과 동시성

JVM이 메모리를 어떻게 쓰는지, 여러 스레드가 어떻게 만나는지.

  • JVM 메모리 구조 — heap·stack·metaspace, GC 개요 객체 생성과 참조 흐름, Young/Old 분리와 generational hypothesis, OOM의 원인.
  • 멀티스레딩 기초 — synchronized·volatile·ExecutorService Thread/Runnable, race condition·deadlock, wait/notify, CompletableFuture 개요.

🔬 심화 / 상세

핵심 페이지에서 분리한 깊이 있는 주제들. 면접에서 단독으로 깊이 묻는 항목들이라 페이지를 따로 둡니다. (작성 예정)

JVM 내부와 GC

  • JVM 내부 구조 심층 작성 예정 class loader, JIT, tiered compilation, escape analysis.
  • GC 알고리즘 심층 작성 예정 Serial / Parallel / G1 / ZGC / Shenandoah, generational hypothesis, pause time tuning.

동시성 심화

  • Java Memory Model(JMM) 작성 예정 happens-before, visibility, atomicity, reordering, volatile/synchronized 의미.
  • 동시성 유틸리티 작성 예정 Lock / ReentrantLock, Semaphore, CountDownLatch, CyclicBarrier, BlockingQueue.
  • ConcurrentHashMap 내부 작성 예정 CAS, bin locking, tree bin, resize 동작.
  • CompletableFuture 심층 작성 예정 비동기 체이닝, 예외 처리, executor 선택, thread pool 고갈.

고급 기능과 운영

  • Reflection / Annotation / Proxy 작성 예정 Spring AOP·DI의 기반, dynamic proxy vs CGLIB.
  • NIO / Netty 입문 작성 예정 blocking I/O vs non-blocking I/O, Selector, ByteBuffer, event loop.
  • Java 성능 튜닝 작성 예정 JFR / JMC, heap dump, thread dump, allocation profiling.
